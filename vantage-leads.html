Instrucciones — Google Apps Script (una sola vez)

Para que esta herramienta escriba en tu sheet, necesitás publicar un Web App en Google Apps Script:

1. Abrí tu Google Sheet → Extensiones → Apps Script

2. Pegá el código del script (pedímelo en el chat)

3. Publicá como Web App → Ejecutar como: Tú → Acceso: Cualquier usuario

4. Copiá la URL que te da y pegala en el código de esta herramienta (variable APPS_SCRIPT_URL)
